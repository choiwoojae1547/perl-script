# Perl 사용자 접속 로그 분석 스크립트

이 저장소는 Perl 해시(Hash) 자료구조를 활용하여 사용자 접속 로그를 분석하는 간단한 스크립트입니다.  
로그 파일에서 사용자별 접속 횟수를 계산하고, 정렬된 결과를 출력합니다.

---

## 🛠️ 사용 기술

- Perl
- 정규표현식(Regex)
- 해시(Hash)
- 파일 입력 처리 (`<>`)
- 조건문 (`if`)
- 루프문 (`while`, `foreach`)
- 삼항 연산자 및 기본값 연산자 (`//`, `? :`)

---

## 📂 예제 로그 파일: `access.log`
